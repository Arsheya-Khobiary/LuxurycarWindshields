## Step 4 — Technical SEO checklist

### Schema (implemented baseline in `index.html`)

- **AutomotiveBusiness** (or `LocalBusiness` if you prefer) + `areaServed` London / Greater London.  
- **WebSite** linked to publisher.  
- **Service** entities for core offers.  
- **FAQPage** for rich results (keep answers factual; match on-page FAQ text **exactly**).

**When you add NAP:** If you get a genuine office or registered address, add full `PostalAddress` and align it with Google Business Profile.

### Image SEO

- Filenames: `mobile-windscreen-replacement-london-van.jpg`, `stone-chip-repair-windscreen-before.jpg`.  
- Alt: describe the image + service + London where natural: `Mobile windscreen replacement technician fitting glass in London`.  
- Compress WebP/AVIF; **LCP** image: priority load, explicit dimensions.

### Core Web Vitals (practical)

- Minimise JS on landing routes; lazy-load below-fold galleries.  
- Preload critical font / hero image only.  
- Avoid layout shift: width/height on images and reserved space for embeds.

### Mobile-first

- Tap targets ≥ 44px for **Call** and **WhatsApp**.  
- Click-to-call `tel:` in header + sticky mobile bar (when you implement multi-page).  
- Form fields: numeric keyboard for phone/reg.
